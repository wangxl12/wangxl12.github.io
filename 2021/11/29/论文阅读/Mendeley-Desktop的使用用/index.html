Mendeley Desktop的使用

发表于 2021-11-29 更新于 2021-11-30 分类于 论文阅读 阅读次数： Valine：

当我们写一篇论文的时候，会用到很多的参考文献，使用论文管理工具可以更加方便的帮助我们提高工作效率，本文介绍使用Mendeley。

为什么选Mendeley?

可以参考这篇文章：五款常用的文献管理神软件

如何使用Mendeley?

下载Mendeley

注意下载Mendeley Desktop，而不是Mendeley Reference Manager，这是两个不同的软件，后者很鸡肋，难用，功能少的可怜，很多人以为后者是前者的新版，其实不然，可以参考这里。对于Mendeley Desktop的下载网址点击这里

安装插件

下载好了Mendeley Desktop(以下都称为Mendeley)，提示安装两个插件：

我就来到office6目录下创建一个名为startup的目录，然后重新点击install就发现在这个目录下新增了这个文件：

然后将这个文件剪切到C:\Users\Admin\AppData\Roaming\Microsoft\Word\STARTUP目录下，重新打开一个word文档，就可以发现在 引用 栏中新增了这一项：

如果要在word中插入一个引用，先点击Insert Citation ，然后出现这个窗口：

点击Go To Mendeley 。

选中一篇论文，点击左上角的Cite 。

然后在word 文档里点击Insert Bibliography，就OK了。

如果数据库中有文献改动，改完了之后点击Refresh即可。

导入文件

导入文件的方式有多种，可以拖拽文件，可以点击Add 添加单个文件、多个文件、文件夹等。

添加GB/T 7714 style

点击Style里的More Styles：

点击Get More Styles ，搜索GB/T 7714-2015 ，然后安装numeric的那个：

安装之后需要在Installed选项卡中选择输入GB搜索，然后选择这个style。一开始的时候会出现英文人名全部大写的情况，可以参考这篇博客修改：

mendeley引入并修改GB/T 7714-2005中文标准参考文献 ，解决人名全部大写问题

创建小组

点击左边的Group 就可以创建小组，为小组命名->邀请组员即可创建成功。Group的文献添加和个人使用时的文献添加是一样的。
